# LameForAndroid
这里为大家讲述，48K采样率，128比特率的Wav音频转MP3，当然也是支持其他采样率音频转码的，需要你自己增加变量，内置了完整的Lame转码库，懂C的完全可以做出一个全能的音频格式转换器，这里只举例一种情况，第一是为了给大家提供一条思路，第二也是为了抛砖引玉！！<br>
